# ganesha

Your life coach who will always be with you to guide you.

Ganesha is an application made for the `Gemini API Developer Competition`  and also has a [Python Api](https://github.com/yesilOguz/ganesha)

Ganesha is a software that aims to guide people by using the power of Gemini. The project aims to guide you in your life and make your life easier by providing you with life coaching. It listens to your conversations to get to know you and it also has endpoints so you can chat with it.

## Some Key features
- [x] Can record voice and send recorded voices to api even if app minimized
- [x] If there is a problem with the server or the user's internet, only the sound is recorded and when the internet comes back, all recorded sounds are sent. 
- [x] Checks server's status, there is a screen if server is down
- [x] Checks jwt's status in every api request, if it is not valid anymore application trying to refresh it
- [x] There is an admin panel for model management ( add model - remove model )
